🐍 Learn Python by Coding — Group Python Quests

Welcome to the Group Python Quests Repository.
This project was created to strengthen foundational programming skills through a structured set of practical Python challenges divided into 6 progressive levels.

The quests focus on:

* Python fundamentals
* Logical reasoning
* Problem-solving
* Functions and loops
* User interaction
* Git & GitHub collaboration

⸻

🎯 Project Goal

The goal of these quests is to build a strong foundation in programming by solving progressively challenging exercises using Python.

Before starting, ensure Python is installed on your system:

python --version

or

python3 --version

📌 Deliverables

✅ Deliverable 1

Each group member must:

* Write Python scripts for quests
* Push their work to the GitHub repository
* Attempt at least 4 tasks from each of the 6 levels

✅ Deliverable 2

Each group must:

* Review two assigned peer groups
* Provide constructive and effective feedback
* Evaluate solutions and repository structure
